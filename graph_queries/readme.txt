aeromatch_search - Implements the airline search engine using PySpark

neo4j_bfs - Implements the shortestPath algorithm from Neo4j directly in Neo4j

pairs - Represents the pairs of source/destination airports used to evaluate the runtime of each model. The pairs are specified using the IDs that Neo4j 
	internally creates.

airportID_pairs - This is a tranformation of the pairs.csv data from the internal Neo4j ids to the airportIDs.
